Estrutura do Banco de Dados (SQL Server)

Este diretório contém o script necessário para reconstruir o banco de dados do projeto.

Como restaurar o banco:
1. Abra o SQL Server Management Studio (SSMS),
2. Crie um novo banco de dados chamado `StatixUPX3`.
3. Abra o arquivo `script_banco.sql` contido nesta pasta.
4. Execute o script para gerar todas as tabelas e relacionamentos.
